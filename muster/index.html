Muster Application

This application is used for analyzing Window's event viewer export to locate sign-on and sign-off events to calculate many detail e.g. Check-in / check-out time, Time spent on screen, etc.

Step 1

Event Viewer Filter

Click on below button to view step by step instructions to search for logon and logoff events in Window's event viewer. Instructions

Make sure to save filter with suitable name e.g. LogonLogoffEvents.

Step 2

Export Filtered Events as CSV

Double click on saved filter to view all events. On right hand side locate option "Save all Events as..." and click on it.

In the "Save as type:" field select "CSV (Comma Separated Value)(*.CSV)"

Click in the "File name:" box and type a file name and save location for the file. For example: "c:\EventViewerExport" will save the file called LogonLogoffEvents.csv

Click "Save" to save filtered events in the CSV file

NOTE: Writing filtered events to csv file may take some time.
